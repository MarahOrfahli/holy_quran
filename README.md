# holy_quran
##  خطوات تشغيل المشروع للمطورين في الفريق
1. `git clone <repository url>`
2. `npm install`
3. `npm run dev`

##  ملفات البيانات (JSON)
هذه الملفات غير موجودة في المستودع نظراً لكبر حجمها.
- قم بتحميل المجلد المضغوط 
- `surah.json` , `surah` , `translation`
- بعد فك الضغط، ضع المجلد في المسار: `src/data/`
  ##  لتشغيله على server
  استخدم التعليمة التالية: 
`serve`
في حال لم يعمل ويتولد رايط محلي يرجى تشغيل الخادم عن طريق
`npm install -g serve`
`serve`
أو
`npx serve .`
